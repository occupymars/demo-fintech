How to trigger events

Click on a user below to view their details and trigger Fourbyfour events. Each event triggers an automated workflow:

  • payment.due → Payment reminder workflow
  • kyc.incomplete → KYC completion workflow
  • transaction.failed → Failed payment recovery workflow
  • account.dormant → Reactivation workflow
